This is a Python script that creates a retro-style radar display using pygame.

It fetches live ADS-B flight data from opensky-network.org
It shows aircraft as blips on a radar screen
It draws a sidebar with status and selected plane info
You can click a plane to get more details, drag to pan, and zoom with mouse wheel or keys

Click a plane to fetch detailed info
Drag the radar to pan the view
Mouse wheel or + / - to zoom
Arrow keys pan, R resets to current location. >_<
Side Note: you need pygame and requests to run it.....
